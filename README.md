# 🐧 Wiki-Colaborativa---LINUX 🐧
  
## **Este é um mini-dicionário com 30 comandos do Linux, utilizando linguagem Markdown**

Com este dicionário, você irá conhecer alguns dos comandos mais usados para lidar com processos, gerenciamento de usuários e entender melhor o funcionamento do sistema. Se você já tem experiência na área ou está apenas começando, aprender e lembrar esses comandos irá fazer uma grande diferença. Eles ajudam a gerenciar servidores com mais eficiência, criar scripts e resolver problemas com mais facilidade.

## **Navegação em arquivos e pastas**  
| Comando | Função                                  | Alguns exemplos na prática                                                   |
|:-------:|:----------------------------------------|:-----------------------------------------------------------------------------|
| **ls**  | Lista o conteúdo de um diretório        | Listar, verificar tamanho e dados técnicos de um arquivo                     |
| **pwd** | Mostra o caminho do diretório atual     | Referência p/ caminho de diretório, localização nas pastas                   |
| **cd**  | Troca de diretório                      | Voltar ou mudar p/ um diretório específico                                   |
| **find**| Encontra um arquivo no diretório atual  | Localização de arquivos com uma característica específica (tamanho,tipo, etc |

## **Operações com arquivos e pastas**
| Comando            | Função                                               | Alguns exemplos na prática                                                  | 
|:------------------:|:-----------------------------------------------------|:----------------------------------------------------------------------------|
| **mkdir**          | Cria um diretório                                    | Criar um diretório ou vários de uma vez, criar subpastas                    |
| **rmdir**          | Remove um diretório                                  | Limpeza e organização de pastas                                             |
| **rm**             | Exclui um arquivo                                    | Excluir ou forçar a remoção de um arquivo ou diretório                      |
| **cp**             | Copia um arquivo para outro local                    | Copiar arquivos ou diretórios de uma origem para outra                      |
| **mv**             | Move ou renomeia um arquivo ou pasta                 | Mover ou renomear arquivos ou diretórios                                    |
| **zip e unzip**    | Compacta ou extrai um arquivo ZIP                    | Auxilia na transferência, compatibilidade e redução de tamanho de arquivos  |
| **cat**            | Exibe o conteúdo de um arquivo                       | Juntar ou visualizar arquivos, numeração e verificação de texto             |
| **cut**            | Seleciona e imprime partes específicas de um arquivo | Lê o texto de um arquivo e retorna uma coluna com uma seção específica      |
| **du**             | Mostra espaço ocupado por arquivos no diretório      | Verificar e monitorar uso de espaço de arquivos e diretórios                | 
| **In**             | Cria um link para diretório ou arquivo               | Hard Link ou Soft Link para organização e economia de espaço                |

## **Gerenciamento de usuários e permissões**
| Comando             | Função                                           | Alguns exemplos na prática                                        |
|:-------------------:|:-------------------------------------------------|:------------------------------------------------------------------|
| **su**              | Troca de usuário dentro da mesma sessão terminal | Acesso à conta _root_, e gerenciar sistema sem _sudo_             |
| **useradd\userdel** | Adiciona ou remove um usuário                    | Controle de scripts de automação ou remoção de usuários           |
| **passwd**          | Define uma nova senha para um usuário            | Criar, alterar ou redefinir uma senha própria ou de outro usuário |

## **Comandos diversos**
| Comando             | Função                                             | Alguns exemplos na prática                                                      |
|:-------------------:|:---------------------------------------------------|:--------------------------------------------------------------------------------|
| **date**            | Visualizar/alterar a hora e data do sistema        | Ajustes do sistema, automação de tarefas                                        |
| **df**              | Mostra espaço livre e ocupado nas partições        | Visualização de espaço livre, gerenciamento de servidores                       |
| **free**            | Mostra a quantidade da memória RAM que está em uso | Diagnóstico de performance, gerenciamento de recursos                           |
| **grep**            | Procura uma palavra dentro de um arquivo           | Filtrar ocorrências, buscar palavra em arquivos ou diretórios                   |
| **head**            | Exibe as linhas iniciais de um texto               | Verificar cabeçalhos, verificação rápida de arquivos                            |
| **nl**              | Exibe o número de linhas e o conteúdo do arquivo   | Localização de linhas específicas, revisão de código (facilita a localização)   |
| **touch**           | Cria um arquivo vazio ou atualiza a data           | Criação de arquivos-teste, automação                                            |
| **sort**            | Organiza as linhas de um arquivo de texto          | Ordenar logs de diversas formas, organização de arquivos e dados                |
| **time**            | Mostra o tempo para concluir um processo           | Análise e diagnóstico de perfomance                                             |
| **uptime**          | Mostra a quanto tempo o sistema está ligado        | Verificar reinicializações e a quantidade de tempo operacional do servidor      |
| **reboot**          | Reinicia o sistema                                 | Atualizações ou lentidão no sistema                                             |
| **shutdown**        | Desliga o sistema                                  | Desligamento do sistema de forma segura, pode ser imediato ou agendado          |
| **clear**           | Limpa a tela do terminal                           | Limpar a tela poluída ou por conta de um comando incorreto                      |

## **Conclusão**
Dominar os comandos básicos do Linux é essencial para a utilização e gerenciamento do sistema, melhorando a experiência do usuário. Este dicionário irá ajudar na utilização do sistema operacional, aumentando a eficácia e diminuindo o tempo gasto na resolução de problemas como um profissional de TI. Com a prática, esses comandos começam a ser utilizados de forma natural, sem precisar ficar checando um comando para cada problema encontrado. A prática leva à perfeição!
